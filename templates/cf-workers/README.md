# Rajt λ - Cloudflare Workers Template

This template should help get you started developing Rajt on Cloudflare Workers.

## Install dependencies

To install, run:

```bash
bun i
```

## Tests

To perform the tests, run:

```bash
bun test
```

## Development environment

Test the application locally in an environment identical to production using the command:

```bash
bun dev
# or
bun run dev
```

## Production environment

Perform the build for production environment using the command:

```bash
bun run build
```
